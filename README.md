# 🧮 Calculator Web Application

## Overview

The **Calculator Web Application** is a simple and responsive calculator built using **HTML, CSS, and JavaScript**. It performs basic arithmetic operations through an intuitive user interface and also supports keyboard input for improved usability.

This project was created to strengthen front-end web development skills by combining responsive design, styling, and JavaScript event handling.

---

## Features

* Responsive and user-friendly interface
* Perform basic arithmetic operations:

  * Addition (+)
  * Subtraction (−)
  * Multiplication (×)
  * Division (÷)
  * Percentage (%)
* Decimal number support
* Clear (C) button to reset calculations
* Backspace (⌫) button to delete the last character
* Keyboard support (numbers, operators, Enter, Backspace, Escape)
* Modern blue-themed UI with hover animations

---

* JavaScript (Vanilla JS)

## How to Run the Project

1. Download or clone the repository.
2. Open `calculator.html` in any modern web browser.
3. Use the on-screen buttons or your keyboard to perform calculations.

---

## How It Works

* Users enter numbers and operators using the calculator buttons or keyboard.
* JavaScript dynamically updates the display.
* Clicking the **=** button (or pressing **Enter**) evaluates the expression and displays the result.
* Invalid expressions are handled gracefully by displaying an **Error** message.
